# HOME BURGERS BACKEND

## Dev


1. clonar el proyecto 
2. ejecutar ``npm install`` p ``yarn install``
3. mirar el env.template y reemplazar con lo que debe ir en el .env para conectarse
4. para que corra el proyecto usamos `yarn start:dev`


## tener en cuenta el docker-compose.yaml
1. debemos de escribir como el .env pero para el docker para poder levantar lo

## Levantar docker

1. escribimos ``docker compose up -d`` y ya se subiria la imagen en docker
